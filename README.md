# MVW-Gallery

## Prerequisites
- run `npm install`
- install [GraphicsMagick](http://www.graphicsmagick.org/)
- create input folder with `./input/YEAR/Gallery/image.ext`
  - rotate (EXIF JPG Autorotate)
  - sort
- place `galleries.json` and `state.json` in `./input`
- run `npm run convert`
